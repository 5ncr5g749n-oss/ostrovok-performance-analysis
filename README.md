# Ostrovok

## Описание
Проект с аналитическими материалами и артефактами исследования.  
Добавлен файл `design.json` с высокоуровневыми гайдлайнами дизайн-системы на основе референсного UI (дашборд в стиле modern SaaS).

## Возможности
- Структурированное описание визуального стиля интерфейса в `design.json`.
- Единые правила по сетке, отступам, типографике и цветам.
- Рекомендации по ключевым UI-компонентам: sidebar, tabs, KPI-карточки, table, filters, badges, buttons.
- Принципы интеракций, состояний, адаптивности и чеклист для генерации интерфейсов ИИ.
